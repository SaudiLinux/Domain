أداة Domain - أداة استطلاع متقدمة للنطاقات

المؤلف: SayerLinux
البريد الإلكتروني: SayerLinux@gmail.com

=== طريقة التثبيت ===

1. تأكد من وجود:
   - نظام تشغيل لينكس
   - بايثون 3.8 أو أحدث
   - مدير الحزم pip

2. قم بتثبيت المتطلبات:
   pip install -r requirements.txt

=== طريقة الاستخدام ===

1. الفحص الأساسي:
   ./domain.py -d example.com

2. فحص كامل مع جميع الوحدات:
   ./domain.py -d example.com --full

3. فحص وحدات محددة:
   # معلومات النطاق
   ./domain.py -d example.com --info

   # استخراج النطاقات الفرعية
   ./domain.py -d example.com --subdomains

   # اكتشاف الروابط المخفية
   ./domain.py -d example.com --urls

   # البحث عن صفحات الإدارة
   ./domain.py -d example.com --admin-finder

   # تحليل سطح الهجوم
   ./domain.py -d example.com --attack-surface

   # فحص الثغرات
   ./domain.py -d example.com --vulns

4. خيارات متقدمة:
   # تعيين عدد مسارات التنفيذ المتوازية
   ./domain.py -d example.com --full --threads 20

   # تعيين مهلة الاتصال
   ./domain.py -d example.com --full --timeout 60

   # استخدام قائمة كلمات مخصصة
   ./domain.py -d example.com --full --wordlist /path/to/wordlist.txt

   # تعيين وكيل المستخدم
   ./domain.py -d example.com --full --user-agent "Mozilla/5.0"

   # تأخير بين الطلبات
   ./domain.py -d example.com --full --delay 1.0

=== المخرجات ===

يتم حفظ النتائج في مجلد 'results' بالتنسيقات التالية:
- تقرير Markdown: domain_YYYYMMDD_HHMMSS.md
- بيانات JSON: domain_YYYYMMDD_HHMMSS.json

=== تنبيه هام ===

هذه الأداة مخصصة لأغراض الاختبار الأمني القانوني والبحث فقط. يجب على المستخدمين الحصول على التصريح المناسب قبل فحص أي نطاقات لا يملكونها. المؤلف غير مسؤول عن أي سوء استخدام أو ضرر ناتج عن هذه الأداة.

=== المميزات ===

1. جمع معلومات النطاق:
   - عناوين IP وسجلات DNS
   - معلومات WHOIS
   - تفاصيل شهادة SSL
   - رؤوس HTTP والتقنيات المستخدمة

2. استخراج النطاقات الفرعية:
   - استخراج سلبي من مصادر متعددة
   - استخراج DNS نشط
   - اكتشاف بالقوة الغاشمة

3. اكتشاف الروابط المخفية:
   - زحف الويب
   - اكتشاف المجلدات والملفات
   - اكتشاف المعلمات

4. البحث عن صفحات الإدارة:
   - اكتشاف لوحات الإدارة الشائعة
   - دعم قوائم الكلمات المخصصة
   - فحص تجاوز المصادقة

5. إدارة سطح الهجوم:
   - فحص المنافذ وتحديد الخدمات
   - تحليل رؤوس الأمان
   - اكتشاف مجموعة التقنيات
   - تحديد CDN و WAF

6. فحص الثغرات:
   - اكتشاف حقن SQL
   - اكتشاف XSS
   - ثغرات تضمين الملفات
   - SSRF
   - الأخطاء في التكوين الأمني

7. إنشاء التقارير التلقائية:
   - تقارير مفصلة بتنسيق Markdown
   - تصدير JSON للأتمتة
   - عرض النتائج بشكل مرئي
